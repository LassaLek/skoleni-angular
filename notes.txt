App
jeden Feature module

Command:
(more in docs: https://angular.io/cli/generate)
ng generate component <name> (can be shortened to g, c)
- some options (other will be explained later)
--export=true|false
When true, the declaring NgModule exports this component.
Default: false
--flat=true|false
When true, creates the new files at the top level of the current project.
Default: false
--module=module
The declaring NgModule.
Aliases: -m

Examples:
- base component
Command: ng g c components/box - add new component BOX to main app module
- to specific module
Command: ng g c feature/components/list --module=feature --export=true - add new component LIST to feature module
- inline component
pregenerated <app-inline-box></app-inline-box>
